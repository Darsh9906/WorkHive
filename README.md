# 🚀 WorkHive

WorkHive is a simple Employee Management System built with **React.js** and **Tailwind CSS**.

The project allows an Admin to assign tasks to employees and lets employees view their assigned tasks through their dashboard.

This project was built mainly to learn and practice React concepts such as state management, Context API, props, localStorage, conditional rendering, and working with nested data structures.

---

## ✨ Features

### Admin

* Admin Login
* Create and assign tasks to employees
* View admin dashboard
* Persistent login using localStorage

### Employee

* Employee Login
* View assigned tasks
* View task statistics:

  * Active Tasks
  * New Tasks
  * Completed Tasks
  * Failed Tasks
* Persistent login after page refresh

## ⚙️ Installation

Clone the repository:

```bash
git clone <your-repository-url>
```

Go to the project folder:

```bash
cd workhive
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
